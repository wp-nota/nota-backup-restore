# Nota Backup & Restore

Nota Backup & Restore is a WordPress plugin for creating full site backups, downloading them as ZIP files, and restoring your site when needed.

## Features

- **Full site backup** — files + database in a single ZIP
- **Database only** backup
- **Files only** backup
- **Download as ZIP** — save backup to your computer
- **Installer PHP** — restore your site on any server without WordPress
- **Restore wizard** — restore directly from the admin panel *(Pro)*
- **Cloud storage** — Google Drive, Amazon S3, Wasabi, Dropbox, OneDrive, FTP/SFTP *(Pro)*
- **Scheduled backups** — daily, weekly, monthly *(Pro)*
- **Email notifications** — success and failure alerts *(Pro)*
- **AES-256 encryption** *(Pro)*

## Quick Start

1. Install and activate the plugin
2. Go to **Nota Backup** in the WordPress admin menu
3. Click **Start Backup**
4. Download the ZIP file when complete
